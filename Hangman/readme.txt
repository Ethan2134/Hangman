HANGMAN

This is game of Hangman created as a class project, using NetBeans. This game was created to play in the NetBeans
IDE, however you can also run in from the command line. It is filled with random words to play with, and you have 
8 guesses to finish the word, otherwise the game ends and you lose. To play, download the project and open the command line.
Navigate to the 'dist' folder and type the following command without the quotation marks: "java -jar Hangman.jar". The souce code is also included,
allowing you to add your own custom words. In order to do so, locate and open the hangman java file. Find the method 'gameLoop', and
there is a comment marking where it should be added. Simply follow the pattern shown in the code and add your own word in the quotation marks.
